Recomendação: Adotar S3 Tables como padrão para novas tabelas Iceberg. O principal benefício é a eliminação completa do overhead operacional de manutenção, com performance de query equivalente ou superior. A economia projetada de ~$31K/ano vem principalmente da redução de horas de engenharia em tarefas de manutenção.
